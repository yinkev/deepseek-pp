# Grok-Style Workflow Visual

Self-contained Grok-inspired HTML/SVG/CSS operations map for the current Oracle governor workflow.

## Open

```sh
open /Users/kyin/Projects/deepseek-pp/.ai-visuals/interactive/index.html
```

No build step or network access is required. The file uses inline CSS, SVG, and JavaScript only.

## Controls

- `Pause motion`: pauses animated paths and particles.
- `Governor focus`: dims the local solver path.
- `Solver focus`: dims the Pro governor path.
