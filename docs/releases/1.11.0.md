## DeepSeek++ 1.11.0

1.11.0 完成一轮面向 PC 浏览器的稳定性与兼容性重构，重点改善长期使用中的数据安全、同步恢复、自动化可靠性、聊天与工具运行稳定性，以及日常加载性能。

### 主要变化

- **聚焦 PC 浏览器**：当前正式支持 Chrome、Edge 和 Firefox；Android 模板、构建脚本和移动端支持入口已移除，不再提供 Android 安装包。
- **本地数据与升级兼容**：记忆、Skill、预设、项目、保存项、MCP、自动化、用量和工具历史使用更明确的版本与校验规则；继续沿用现有存储标识，损坏数据或未知未来版本会明确报错，不会静默覆盖原始内容。
- **同步与自动化恢复**：同步上传采用完整代次发布，下载先暂存并支持失败回滚；未完成操作可在重启后恢复。自动化任务补强取消、执行租约和重复触发防护，降低后台任务残留或重复执行的概率。
- **聊天、工具与本机集成更稳**：收紧运行时消息、DeepSeek 请求、流式工具调用、MCP 传输和 Native Host 的输入边界；插件更新或页面重载后的悬浮聊天、工具状态和错误反馈更可预测。
- **减少日常性能开销**：合并高频存储写入，集中管理页面监听与生命周期，按需加载侧边栏页面、内置 Skill 资源和 Python 运行时；首个 Chat 页面构建体积约下降 23%，Chat 路由约下降 84%，并增加关键分包和离线资源发布预算校验。
- **浏览器权限说明**：Chrome 和 Edge 的权限集合不变；Firefox 增加 `identity`，仅用于用户主动启用 Google Drive 或 OneDrive 同步时的 OAuth 登录。

### Chrome Web Store 更新说明

中文：

- 加强记忆、Skill、预设、自动化和同步数据的校验与故障恢复，并继续兼容现有本地数据。
- 提升定时任务、聊天、工具调用、MCP 连接以及插件更新后的运行稳定性。
- 减少重复存储写入，并按需加载较重的页面和资源，降低日常运行开销。
- 当前正式支持 PC 端 Chrome、Edge 和 Firefox，不再提供 Android 版本。
- Chrome 版本不新增浏览器权限。

English:

- Strengthens validation and recovery for memories, Skills, presets, automation, and sync data while preserving existing local data.
- Improves reliability for scheduled tasks, chat, tool calls, MCP connections, and extension reloads.
- Reduces redundant storage writes and loads heavier pages and resources on demand.
- Official support now focuses on Chrome, Edge, and Firefox on PC; Android is no longer provided.
- The Chrome package adds no new browser permissions.
