## DeepSeek++ 1.0.3

1.0.3 是云同步、提示词复用和 MCP 连接增强版本，重点让保存项可以更快进入对话，让同步来源扩展到 Google Drive / OneDrive，并提升本机 Shell 与 Streamable HTTP MCP 的可靠性。

### 主要变化

- **保存项插入**：保存页和资料库中的常用 prompt、片段和书签可直接插入当前聊天输入框，复用固定指令和工作流更顺手。
- **云同步来源**：同步设置新增 Google Drive 和 OneDrive，继续保留 WebDAV；Drive / OneDrive 使用用户自己的 OAuth 应用配置，数据只在用户启用同步后写入对应云端应用空间。
- **MCP 连接**：MCP HTTP 传输支持 Streamable HTTP 会话，远程工具服务的发现、连接和执行状态更稳定。
- **Shell 安全边界**：Shell MCP 执行环境改为最小环境变量集，减少主机进程里的敏感变量被本机命令继承；Windows PowerShell 持久会话也更稳定。
- **同步兼容性**：GitHub Skill 从仓库根目录导入后，跨设备同步下载不再因为空 rootPath 校验失败。
- **回归覆盖**：新增保存项插入、云同步后端、MCP 传输策略、GitHub Skill 同步校验和 Shell 环境隔离测试。

感谢本版本贡献者：[@maoxin1234](https://github.com/maoxin1234) 改进 Shell MCP 环境隔离与 Windows 会话稳定性。

### Chrome Web Store 更新说明

中文：

- 保存的 prompt、片段和书签现在可直接插入当前聊天输入框，复用常用指令更快。
- 同步设置新增 Google Drive 和 OneDrive，并继续支持 WebDAV；Drive / OneDrive 仅在用户启用同步并配置自己的 OAuth 应用后使用。
- MCP HTTP 连接支持 Streamable HTTP 会话，远程工具发现和执行状态更稳定。

English:

- Saved prompts, snippets, and bookmarks can now be inserted directly into the current chat input for faster reuse.
- Sync settings now add Google Drive and OneDrive while keeping WebDAV; Drive / OneDrive are used only after the user enables sync and configures their own OAuth app.
- MCP HTTP connections support Streamable HTTP sessions for more reliable remote tool discovery and execution state.
- Shell MCP now uses a smaller local command environment to reduce sensitive environment-variable inheritance, with better Windows PowerShell persistent sessions.
